# SPTENAMARK

## Overview
SPTENAMARK is a centralized online pharmacy marketplace designed for Ethiopia Pharmacies.  
It connects licensed pharmacies with customers, making it easy to buy medicines and cosmetics online.  
The platform charges a small fee per transaction, ensuring sustainable growth.

## Problem
- Lack of organized online pharmacy systems in Ethiopia.
- Customers struggle to find medicines and cosmetics easily.
- Pharmacies lack digital tools to manage inventory and sales.

## Solution
- Unified marketplace for pharmacies and customers.
- Secure user registration and pharmacy verification.
- Transparent pricing and commission-based revenue model.

## Features
- Customer product catalog with search & filters.
- Shopping cart and checkout flow.
- Pharmacy dashboard for product management.
- User registration/login with order history.

## Tech Stack
- Frontend:html,Js,Css
- Backend: Node.js + Express (future)
- Database: Firebase / Supabase / MongoDB Atlas (cloud-based)

## Roadmap
1. Frontend development (catalog, cart, login).
2. Backend APIs (authentication, product CRUD, transactions).
3. Deployment & pilot testing with Addis Ababa pharmacies.
4. Mobile app (React Native/Flutter).
